# HY3D Benchmark Inputs

Use this folder for small, non-private benchmark image placeholders only.
Do not commit heavy real images, private captures, generated outputs, or engine workspaces.

Recommended real-image coverage for local benchmark runs:

1. Solid simple object.
2. Figure, character, or toy.
3. Object with a handle.
4. Open container.
5. Thin structure.
6. Complex surface.
7. Symmetric object.
8. Difficult image with low contrast or complex background.
